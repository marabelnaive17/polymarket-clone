## 🖥️ What this app does

polymarket-clone is a prediction market app. It lets you buy Yes or No shares on real-world events.

It uses:
- Vue 3 for the user screen
- Solidity smart contracts for the rules
- Java for app support
- Polygon for on-chain activity
- USDT for escrow and settlement

This app is built for users who want a simple way to take part in event markets on Windows.

## 🎯 How to use the app

### Browse markets
Open the main page and look through the list of events. Each market shows a question, such as a real-world outcome.

### Buy Yes or No shares
Choose:
- Yes if you think the event will happen
- No if you think the event will not happen

The app shows the share price before you confirm.

### Review your position
After you buy shares, you can check:
- What you own
- The current market status
- Your value in the market

### Settle results
When the event ends, the smart contract handles settlement based on the market result.

## 🔐 Wallet and network setup

This app uses Polygon and on-chain USDT escrow.

### You may need
- A Polygon wallet
- Some USDT in your wallet
- Enough network balance for gas fees

### If your wallet is not connected
1. Open the wallet app
2. Make sure you are on the Polygon network
3. Confirm the connection request
4. Refresh the polymarket-clone app

### If the network is wrong
Switch your wallet to Polygon before you try again.
